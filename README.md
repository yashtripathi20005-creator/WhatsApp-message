# WhatsApp Message Scheduler

A Python-based automated scheduler for sending WhatsApp messages at scheduled times.

## Features

- Schedule messages to be sent at specific times on specific days
- CLI interface for managing scheduled messages
- Automatic sending using WhatsApp Web
- Persistent storage using JSON
- Logging of all operations

## Requirements

- Python 3.7+
- Google Chrome browser installed
- Active internet connection
- WhatsApp account (must be active on WhatsApp Web)
